# Smart Home Dashboard

A custom smart home dashboard built for ESP32-S3 with LVGL, designed to work with Homey smart home hub.

This project is based on the Waveshare ESP32-S3 RGB LCD example and provides a clean foundation for building a smart home control interface.

All of the code is purely written by AI


# Deploy

To deploy to device rurn command: 

```
esphome run loftdashboard.yaml
```

do see debug logs from the device:

```
esphome logs loftdashboard.yaml
```

and choose option choos option 3 ("Over The Air (waveshare-s3-7.local)")
